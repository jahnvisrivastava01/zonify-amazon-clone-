# 🛍️ Zonify – Amazon Clone

A modern, responsive **Amazon-inspired e-commerce frontend** built using **React, Vite, and Tailwind CSS**.
Zonify delivers a seamless shopping experience with fast performance, clean UI, and dynamic cart functionality.

---
---

## ✨ Features

* 🏠 **Home Page**
  Clean and responsive layout with product listings

* 🛒 **Cart System**
  Add/remove items dynamically with real-time updates

* 🔍 **Product Details Page**
  View detailed product information

* ⚡ **Fast Performance**
  Built with Vite for lightning-fast development and build speed

* 📱 **Responsive Design**
  Optimized for mobile, tablet, and desktop devices

* 🎨 **Modern UI**
  Styled using Tailwind CSS for a sleek and minimal design

---

## 🛠️ Tech Stack

* ⚛️ **React**
* ⚡ **Vite**
* 🎨 **Tailwind CSS**
* 📦 **JavaScript (ES6+)**
